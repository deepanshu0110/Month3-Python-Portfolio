# Month 3 — Python & Data Analytics Portfolio

Part of a structured 12-month data analytics self-learning program.

## What This Covers

**Days 32–62 — Python / Pandas Core**
- Data cleaning and transformation
- Exploratory data analysis (univariate, bivariate)
- Feature engineering
- Matplotlib / Seaborn visualisation
- NumPy arrays and matrix operations
- Pivot tables, window functions, apply/lambda
- Time series basics, MultiIndex, performance
- Notebook storytelling and export

**Days 63–65 — Math Bridge (ML Foundations)**
- Day 63: Matrices and dot products — the engine behind linear regression and neural nets
- Day 64: Gradients and gradient descent — implemented from scratch
- Day 65: Central Limit Theorem — sampling distributions, confidence intervals, bootstrap hypothesis testing

## Tech Stack

Python 3.11 · Pandas · NumPy · Matplotlib · Seaborn · JupyterLab

## Structure

Each day folder contains:
- Raw data (unmodified)
- Practice tasks
- Concept notes
- Answer key
- Scoring rubric
